collection of promotion material (videos, images, gif) for RDM,

collection of use case, user stories (text)

creation of ideas for new video material (text, images)

creation of videos

How will the data be collected or created?


Data will be collected via a github repository issues and emails or via internet browsing

Documentation and Metadata

What documentation and metadata will accompany the data?


metadata scheme still needs to be developed, but at least source of the data (PID, author, internet address) and license which apply will accompany the data.

Ethics and Legal Compliance

How will you manage any ethical issues?


NA

How will you manage copyright and Intellectual Property Rights (IP/IPR) issues?


For material which is not under CC licenses, we will only list links to the material. For the rest, we will indicate the license and cite authors

Storage and Backup

How will the data be stored and backed up during the research?


Data will be stored on Github, data collected via github issues will be integrated in the code. Each collaborator will therefore have a copy of the whole data on a local disk.
At regular time, new versions will be created, which will back up all data on zenodo automatically.

How will you manage access and security?


Access is granted during the project which occurs in the open. All data will therefore be accessible at all time on github and/or zenodo. There is no securtity issue

Selection and Preservation

Which data are of long-term value and should be retained, shared, and/or preserved?


All but the github issue will be preserved and shared.

What is the long-term preservation plan for the dataset?


The dataset will be saved on zenodo.

Data Sharing

How will you share the data?


Data will be available at all time on github and zenodo, under an appropriate license:

CC0 if possible

CC-BY for video content

Necessary license, if a SA licensed was previously used.

Are any restrictions on data sharing required?


Only data which is already licensed will be restricted following their own restriction.
